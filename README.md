# rosquilletes
A simple content management system in PHP.

## Install
Go to your www folder and write:
```
git clone https://github.com/cosmogat/rosquilletes
cd rosquilletes/js
wget https://code.jquery.com/jquery-3.3.1.min.js -O jquery.js
cd -
```
